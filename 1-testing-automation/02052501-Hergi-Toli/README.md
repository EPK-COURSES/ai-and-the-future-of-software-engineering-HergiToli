\## Assignment Overview



This work is part of SWE101 Assignment 1 and Assignment 2. The topic is AI Testing, Debugging, and Automation. The research explains how Artificial Intelligence can help software engineers with testing, finding bugs, debugging errors, and automating repetitive tasks.



The assignment also focuses on the importance of human supervision. Even if AI can generate tests or suggest solutions, software engineers still need to check the results, understand the code, and make final decisions.



\## Main Focus



The main focus of this assignment is to understand how AI can help with:

\- generating test cases

\- detecting bugs

\- explaining errors

\- supporting debugging

\- improving automation and CI/CD workflows

\- reducing repetitive work



\## Conclusion



AI can make testing, debugging, and automation faster, but it cannot fully replace human responsibility. A good software engineer should know how to use AI tools carefully while still understanding the software and checking the results.
